## Visual Direction

Use the uploaded logo palette as the color foundation for the new website.

## Core Palette

- Deep navy background: `#123249`
- Near-black outlines/shadows: `#050A0F`
- Bright cyan text/accent: `#8DEBFF`
- Pale sky blue fill: `#9ADFFF`
- Mint/seafoam accent: `#7FF2C2`
- Soft white highlight: `#F5FFFF`

## Style Notes

- High-contrast, gaming/esports-inspired look.
- Dark navy should be the main background color.
- Cyan and mint should be used for headings, borders, glow accents, buttons, and highlights.
- Use black or near-black strokes/shadows to keep the bold logo-like feel.
