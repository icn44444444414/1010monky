AskHackers – lär dig Python på svenska

En interaktiv kodutbildning där du bygger spelet Snake steg för steg. Koden körs direkt i webbläsaren via Pyodide, ingen installation behövs.

Tanken bakom AskHackers är att de flesta som vill lära sig koda ger upp för att det blir abstrakt för fort. Här bygger man något man känner igen, ett spel, och ser koden köra direkt medan man skriver den. Vi lade mycket tid på att hålla språket vardagligt och stegen små, så att någon som aldrig öppnat en kodredigerare hänger med.

Allt är på svenska, vilket det sällan är när man försöker lära sig programmering. Vår del var att forma upplägget, bygga sidan i Flask och få Pyodide att köra Python tryggt i webbläsaren, så att fokus kan ligga på att lära sig och inte på att installera verktyg.

Flask Python Pyodide
